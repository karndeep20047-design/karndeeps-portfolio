## ✨ About The Project
This repository contains the source code for my personal portfolio. It is designed to showcase my journey as a full-stack developer, highlighting selected projects, my technical capabilities, and my approach to building software. The architecture emphasizes modern client-side routing, seamless 3D interactions, fluid animations, and ultra-clean code.

## 🛠️ Technical Stack

This portfolio is built on a highly modern, performant frontend stack:

*   **Framework**: React 19 + TanStack Start (Nitro server engine)
*   **Styling**: Tailwind CSS v4 + Custom Vanilla CSS for glassmorphism
*   **Animations**: Framer Motion (`motion/react`)
*   **Routing**: TanStack Router (File-based routing)
*   **State Management**: TanStack Query
*   **UI Components**: Radix UI + Custom responsive elements
*   **Tooling**: Vite, TypeScript, ESLint, Prettier
*   **Deployment**: Vercel (Serverless Functions)

## ⚙️ Running Locally

1.  **Clone the repository**
    ```bash
    git clone https://github.com/karndeep20047-design/karndeeps-portfolio.git
    ```
2.  **Install dependencies**
    ```bash
    npm install
    ```
3.  **Run the development server**
    ```bash
    npm run dev
    ```
4.  **Open your browser** and navigate to `http://localhost:5173`.
