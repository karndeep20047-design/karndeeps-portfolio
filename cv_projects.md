# Portfolio Projects - CV Bullet Points

## AfriSpace
- Single-handedly developed a full-stack outdoor advertising marketplace using PHP, MySQL, and Tailwind CSS, facilitating live billboard bookings and automated PDF invoicing.
- Integrated the Safaricom Daraja API to implement M-Pesa STK Push payments, enabling instantaneous transaction confirmations via callback polling.
- Engineered an interactive geospatial discovery map using Leaflet.js and OpenStreetMap, implementing spatial indexing and data clustering for optimized inventory retrieval.
- Deployed the application to a production environment, configuring the server architecture to securely handle real-time payment webhooks and live GIS data.

## UniScope
- Acted as Scrum Master and Lead Developer for a team of 6, delivering a secure University Magazine CMS utilizing PHP, MySQL, and Bootstrap under strict Agile deadlines.
- Designed a robust Role-Based Access Control (RBAC) architecture with bcrypt authentication and CSRF protection, ensuring secure multi-tenant routing for students, editors, and admins.
- Implemented an automated email notification pipeline via PHPMailer and comprehensive audit logging to track all editorial state changes and maintain data integrity.
- Managed the end-to-end deployment lifecycle, ensuring the application was hosted securely with SSL encryption to protect sensitive university editorial data.

## NexPesa
- Architected and developed a modern personal finance dashboard utilizing React, TypeScript, and Tailwind CSS to create a premium, interactive user interface.
- Integrated Supabase Authentication with strict Row Level Security (RLS) to enforce per-user data isolation and secure backend data modeling on PostgreSQL.
- Built dynamic, interactive financial analytics and goal-tracking features utilizing Recharts, significantly enhancing user engagement with real-time spend visualization.
- Deployed the frontend application seamlessly on Vercel with Cloudflare integration, ensuring high availability, edge-cached performance, and CI/CD automation.

## SentinelPay
- Engineered a robust fintech backend API simulating East African payment infrastructure using Node.js, TypeScript, PostgreSQL, and Prisma ORM.
- Implemented real-time AI fraud detection by integrating the Gemini AI API to asynchronously analyze and flag suspicious transactions based on custom rule engines.
- Ensured zero-loss data integrity by developing atomic wallet transfers using Prisma's transactional rollbacks, alongside simulated M-Pesa STK Push webhooks.
- Containerized the backend infrastructure and deployed the live API and PostgreSQL database on Render, ensuring scalable and reliable cloud hosting.

## SecureAuth API
- Designed and developed a production-grade secure authentication REST API using Node.js, Express, and PostgreSQL, focusing on industry-standard security practices.
- Implemented a dual-token JWT architecture (access and refresh) with refresh tokens securely stored as bcrypt hashes to prevent plain-text database exposure.
- Fortified system security by enforcing strict Role-Based Access Control (RBAC) middleware, Zod input validation, and rate-limiting brute-force protection across all endpoints.
- Fully containerized the application architecture using Docker and Docker Compose, enabling seamless local development and a zero-configuration automated deployment workflow.
